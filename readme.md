## How to Run:

- Clone the project

```bash
  git clone https://github.com/correa-sergio/parsing-html
```

- Go to the project directory

```bash
  cd parsing-html
```
- Set execute permission 

```bash
  chmod +x parsing_html.sh
```
- Run the script 

```bash
./parsing_html.sh
```
- Input

```bash
Usage example: 

Ex: ./parsing_html.sh URL

Ex: ./parsing_html.sh just-an-example.com.br
```
